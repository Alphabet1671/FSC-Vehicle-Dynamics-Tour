# FSC-Vehicle-Dynamics-Tour
 ## 背景
 作于2026年夏

 作为2024、2025赛季FSAE EV参赛队员及2026FSTC裁判，据本人观察，FSC车队在整车动力学方面具有缺乏科学指标体系支撑、缺乏理论知识、缺乏调校经验、缺乏数据采集能力、和缺乏车手反馈几大问题。现有中文大学生方程式整车动力学教程存在适用性不高、重点不明确、系统性不足等问题。为增强FSC赛区平均水平，现启动中国大学生方程式整车动力学巡回论坛计划。
 ## 简介
 该项目分为以下两个部分：
 ### 公开技术教程与工具
项目将详细讲解从整车动力学基本原理、赛车指标体系出发，提供基于python的开源简化动力学模型和.ipynb形式的互动教程。其中将包括以下几个大板块：
- 基于轮荷敏感摩擦圆轮胎模型和GGV的稳态圈速仿真
- 基于FreeCAD及其Python API的运动学设计工作流
- 俯仰平面半车垂向动力学模型的运动方程矩阵推导及特征值阻尼比优化
- 针对整车横摆响应的横滚模态动态优化
 ### 年度巡回测试
鉴于大部分车队车手主观反馈能力欠缺，本人可按需试驾，提供主观反馈。

项目计划于每年暑假进行巡回测试，第一年报名车队将平摊数采系统研发成本并承担项目组1-2人差旅，此后报名车队将仅需要承担差旅
 本项目将提供具有以下功能的独立数据采集及记录设备2-3天使用时间：
 - 实车已有CAN总线记录
 - 地速向量
 - 光学轮速
 - 车身离地高度
 - 轮胎接地姿态及形变
 - 胎温胎压
 - 应变计
 - 空速管
 - 避震位移
 - 提供额外数模转换及数字传感器接口
并将在此期间开展整车概念设计及底盘设计相关答疑活动与技术交流。车队将获得测试过程中所有记录的原始数据。
