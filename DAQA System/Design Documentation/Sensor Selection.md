# 传感器选型
## 记录仪
选用最便宜的多通道CAN SD卡记录仪，需要至少三通道：一通道预留本项目所有传感器，二通道预留赛车主CAN线，三通道备用。
所有模拟传感器信号均就近数模转换为CAN信号，需自研一套高精度CAN ADC。
CAN/RS485数字传感器优先，模拟传感器次之

## 惯导

惯导传感器作为六轴加速度和地速估计的来源，精度极其重要，顾及本项目预算，选用FDISystems开发的Deta40惯导模块。该模块采用Drone CAN通讯，可外接GPS用于卡曼滤波地速估计。

## 角模块
轮边模块需要达成以下功能：
- 六轴相对车身定位
- 应变片数模转换
- 胎温测量
- 胎压测量？

### 六轴定位
由于GPS惯导模块的特性限制及天线距离问题，无法使用。最优解为使用多个单点激光测距模块，通过测量与车身的多点距离来测定立柱的六轴定位。需要量程0.5m，精度0.5mm以下的激光传感器

需要获得以下实时参数：
- 前轮转角
- 轮胎接地角

轮胎接地角可直接使用两个对地激光读取，可以通过夹具调节安装角度

IMU可通过重力加速度定位z轴，如设计夹具可以确保与IMU轴线垂直度，则可以使用IMU获得

### 胎温测量
红外胎温传感器MLX90640或类似红外点阵，RS485/Modbus RTU协议，需轮边模块转换

### 胎压测量
购买市售TPMS后破解协议或直接劫持接收器信号

### 应变片数模转换
淘宝数模RS485/Modbus RTU转换模块
